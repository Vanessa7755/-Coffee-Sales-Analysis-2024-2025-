#  Coffee Sales Analysis 2024-2025
An interactive dashboard which entails sales breakdown, highlighting key drivers (products, months, weekdays and time of day).  
## Data Source
This dataset was acquired from Kaggle and was cleaned and well tabulated using Excel to create analyses and visualizations aimed at business growth.[  <img width="927" height="432" alt="cofee data" src="https://github.com/user-attachments/assets/09e1a9e6-b5e0-48dd-8b14-4981cdacb058" />


## Project Overview
A real-time project based on Kaggle data which showcases an interactive excel dashboard  focusing on sales performance, providing insights into customer trends, product popularity(top performing coffee type by sales and profitability and growth drivers to support business strategies. Tools used were Microsoft Excel, Power Query, Pivot Tables, Charts and Slicers

## Weekday & Hourly Pattern
### Weekday Sales
- The peak sales occur on **Tuesday**  
- Reason: After a busy weekend, Mondays are slow, but Tuesday is the best day for coffee consumption to settle back into routine.  
| Day | Sales |
|------|--------|
| Sun  | 419    |
| Mon  | 544    |
| Tue  | 572    |
| Wed  | 500    |
| Thu  | 510    |
| Fri  | 532    |
| Sat  | 470    |

### Hourly Sales Pattern
- Peak sales hour: **10 am**

## Growth Drivers
- **Marketing:** Implement social media campaigns, ads, and partnerships to increase sales.  
- **Festive Seasons:** Increased sales during October-December, contributing to approximately 45% growth annually due to festive seasons.
